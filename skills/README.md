# board-tui skills

Skills for Claude Code that integrate with board-tui MCP.

## Installation

Symlink each skill into `~/.claude/skills/`:

```bash
ln -s ~/repos/board-tui/skills/task-cards ~/.claude/skills/task-cards
```

Or copy the SKILL.md into an existing skill directory.

## Skills

### task-cards
How to create, read, update, and manage task cards across repos via board-tui MCP tools.

See also: [local-agent](https://github.com/dincamihai/local-agent) ships a `delegate` skill for delegating task cards to a local qwen agent.
